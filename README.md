# ts-freak

Plot frequency of events from a list of timestamps.


## Requirements
- gnuplot

## Usage

~Note: Current expected timestamp format 'YYYY-MM-DD'~

`./tsf <ts_file`

Example usage in a processing pipe:
`sed s/<strips all text after ts>// log_file | ./tsf`
